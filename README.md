# 🍛 Taste Of India

**Taste Of India** is a full-stack web application where users can explore, share, and discover delicious food recipes from across India.
The platform allows anyone to **view recipes publicly**, while registered users can **add their own recipes** and contribute to the community.

All user-submitted recipes are **reviewed and approved by the admin** before becoming visible to the public, ensuring quality and authenticity.

---

## 🚀 Live Status

🟢 The project is currently **in production** and actively maintained.

---

## 🎯 Features

### 👨‍🍳 Public Users

* View all available recipes without login
* Explore different food dishes and cooking methods
* Clean and simple UI for browsing recipes

### 🔐 Registered Users

* Secure user authentication (Login / Register)
* Add new recipes to the platform
* Share cooking steps, ingredients, and food details
* Submitted recipes are sent for **admin approval**

### 🛠 Admin Panel

* Approve or reject submitted recipes
* Manage all recipes and users
* Full control over platform content
* Maintain quality and authenticity of recipes

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/itspksharma/TasteOfIndia.git
```

Go to project directory:

```
cd TasteOfIndia
```

Install dependencies for backend:

```
cd backend
npm install
```

Install dependencies for frontend:

```
cd ../frontend
npm install
```

Run backend:

```
npm start
```

Run frontend:

```
npm start
```
